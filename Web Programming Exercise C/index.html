Exercise C Home Page

This is the index (home) page for "Exercise C", that will act as an access point for a practice website consisting of four folders and a variety of HTML and image files distributed in those folders. Each folder on a website can have an HTML file named index.htm (or .html). The name of this "home page" file depends on the configuration settings of the web server hosting the site. That file acts as a table of contents and link launching point for other files in that folder (or other locations if needed). If this page was stored on a website at the web address http://www.host.net/exercise_c/index.html, most servers would serve up this page even if the user typed only the URL http://www.host.net/exercise_c/ leaving off the filename. Note that it is best practice to include the slash following the final folder label (exercise_c in this example) to indicate that it is a folder name. Most servers will allow this slash to be omitted.
